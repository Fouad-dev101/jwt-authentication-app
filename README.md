# 🔐 JWT Authentication Full-Stack App

A complete authentication system with JWT tokens, httpOnly cookies, and protected routes.

## 🚀 Features

- User Registration & Login
- JWT Authentication with httpOnly Cookies
- Protected Routes
- User Profile Management
- User Directory
- Responsive Design

## 🛠️ Tech Stack

**Backend:** Node.js, Express.js, JWT, bcryptjs ;
**Frontend:** React, React Router, Axios, Context API

## 📦 Installation

### Backend Setup
bash
cd backend
npm install
npm run dev
### Frontend Setup
bash
cd frontend
npm install
npm start
## 🌐 Usage
Open http://localhost:3001

Create an account at /signup

Login at /login

Explore your profile at /profile

View all users at /users
